**必須公開其完整原始碼**，或另行洽談商業授權。

**自願贊助** = 對開源生態的支持，讓本工具能長期維護下去。

This tool is licensed under **AGPL-3.0**: any commercial use that
integrates this tool requires **either** publishing the full source
code of your product, **or** negotiating a separate commercial license.

**Voluntary sponsorship** = supporting the open-source ecosystem so
this tool can be maintained long-term.
